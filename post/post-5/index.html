To-Do Calendar - Day0 訂定目標與題目

  • 2022/04/12
  • 1-minute read
  • SideProject

工作中沒有機會碰到的技術，就自己寫個 Side Project 邊用邊學吧！

題目發想

上班的時候，常常花很多時間在寫工時表，如果當天忘了紀錄，隔幾天在寫腦袋鐵定一片空白，又得花時間重新回憶這幾天都在處理什麼事情。於是想到可以利用我記代辦事項的習慣來解決這個問題，如果能將這些過去的代辦事項存起來，並能在日曆上顯示，就能大大的提升工作效率！

需求分析

最近看了一部日劇《我要準時下班》，在劇中女主角東山結衣分享了她的高效工作法，其中一點很值得參考，就是釐清工作的優先順序，哪些是必須優先處理的工作、哪些可以稍後處理、哪些有空在做就好等等，這樣才能有效管理時間，簡單說就是任務要有區分重要性的功能。


訂定目標

使用 Vue.js + Spring Boot + MongoDB 來開發，並以 RESTful API 串接前後端。

定義功能

  • 帳戶管理：登入、註冊、編輯、個人設定、登出
  • 代辦事項：新增、刪除、編輯（內容/chockbox/拖曳順序/顏色）、查詢
  • 原子習慣：新增、刪除、編輯（內容/highlight/拖曳順序/顏色）、查詢
  • 備忘錄牆：新增、刪除、編輯（內容/拖曳順序/顏色）、查詢
