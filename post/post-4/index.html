安裝 Vue-Cli 遇到的舊版本無法解除安裝問題

  • 2022/04/11
  • 1-minute read
  • 程式筆記

在嘗試安裝 vue-cli 3 時，因過去曾裝過 vue-cli 2.9.6 版本，於是根據官網文件

  • 先解除安裝舊版本

    npm uninstall vue-cli -g 
> up to date in 0.034s

  • 再安裝新的 vue-cli

    npm install -g @vue/cli

然而再運行 vue -V 指令，版號仍然是 2.9.6，也無法使用 vue create 來創建專案，說明舊版本仍沒解除安裝



解決方法

  • 運行 where vue 指令

    where vue
C:\Users\fiber\AppData\Roaming\npm\vue
C:\Users\fiber\AppData\Roaming\npm\vue.cmd

  • 將列出的檔案手動刪除

  • 再運行 vue -V 指令，已無法查看版號

    vue -V 
> 'vue' 不是內部或外部命令、可執行的程式或批次檔。

  • 這時再重新安裝新版本

    npm install -g @vue/cli

  • 再運行 vue -V 指令確認，終於安裝成功啦～ 
    vue -V 
> 3.11.0
