在 Hugo 中使用的 Markdown 語法

  • 2022/04/06
  • 1-minute read
  • 程式筆記

本篇整理了可以在 Hugo 文件中使用的基本 Markdown 語法（呈現會因 Hugo 主題不同而有所差異）：

標題

# H1

## H2

### H3

#### H4

##### H5

###### H6

H1

H2

H3

H4

H5
H6


粗體

**粗體**

粗體

斜體

*斜體*

斜體

粗斜體

***粗斜體***

粗斜體

刪除線

~~刪除線~~

刪除線

引用

> 這裡是一段引用文字

這裡是一段引用文字

無序清單

- 清單
  - 子清單
  • 清單
    • 子清單

有序清單

1. 清單1
2. 清單2
3. 清單3
  1. 清單1
  2. 清單2
  3. 清單3

checkbox

 - [ ] uncheck
 - [x] check
  • uncheck
  • check
